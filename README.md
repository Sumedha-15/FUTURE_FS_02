# 🚀 Mini CRM System

A simple **Client Lead Management System** built using:

* Node.js
* Express.js
* MongoDB (Atlas)
* HTML, CSS, JavaScript

---

## ✨ Features

* ➕ Add new leads
* 📋 View all leads
* 🔄 Update lead status (new → contacted)
* ❌ Delete leads
* 🌐 Connected with MongoDB Atlas

---

## 🛠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Database: MongoDB

---

## ⚙️ How to Run

### 1. Clone the repository

```
git clone https://github.com/Sumedha-15/FUTURE_FS_02.git
```

### 2. Install dependencies

```
cd backend
npm install
```

### 3. Start server

```
node server.js
```

### 4. Open frontend

Open `frontend/index.html` in browser
